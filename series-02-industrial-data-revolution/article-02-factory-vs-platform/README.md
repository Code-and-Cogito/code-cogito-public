# Article 02 — Factory Discipline vs Platform Algorithms

**From Taylor's stopwatch to Uber's algorithm: how worker control evolved.**

Using Python to compare Taylorist factory output distributions with gig economy earnings patterns, and visualize how discipline mechanisms shifted from physical to algorithmic.

## Quick Start

```bash
pip install numpy pandas matplotlib
python factory_vs_platform_analysis.py
```
