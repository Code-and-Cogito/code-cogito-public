# Article 04 — Data Archaeology of the Industrial Revolution

**What do 200-year-old GDP records tell us about today's economy?**

Using Python to reconstruct historical GDP trends, urbanization patterns, and economic structure shifts during the Industrial Revolution.

## Quick Start

```bash
pip install numpy pandas matplotlib
python data_archaeology_analysis.py
```

## What You'll Get

**Charts:**
- `gdp_analysis.png` — GDP per capita trends during industrialization
- `urbanization.png` — Rural-to-urban population shift
- `economic_structure.png` — Economic structure transformation
