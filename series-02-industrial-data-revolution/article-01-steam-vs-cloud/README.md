# Article 01 — Steam Engine vs Cloud Computing

**How two technologies — separated by 200 years — followed the same adoption curve.**

Using Python and the Bass diffusion model to compare steam engine adoption (1760s) with cloud computing adoption (2000s), plus cost structure evolution and market growth analysis.

## Quick Start

```bash
pip install numpy pandas matplotlib scipy
python steam_vs_cloud_analysis.py
```

## What You'll Get

**Terminal output:**
- Technology adoption curve parameters
- Cost comparison across eras
- Market growth trajectory

**Charts:**
- `technology_adoption_bass.png` — Bass diffusion S-curve: steam vs cloud adoption
- `cost_comparison.png` — Cost structure comparison across industrial and digital eras
- `market_growth.png` — Market growth overview
