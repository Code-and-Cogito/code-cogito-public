# Article 05 — Child Labour vs Algorithmic Bias

**The moral blind spots of two revolutions: exploiting children then, encoding bias now.**

Using Python to compare historical child labor patterns with modern algorithmic bias, analyze facial recognition disparities, and draw parallels between two eras of unchecked technological progress.

## Quick Start

```bash
pip install numpy pandas matplotlib
python child_labour_vs_bias_analysis.py
```

## What You'll Get

**Charts:**
- `child_labor.png` — Historical child labor rates and trends
- `facial_recognition_bias.png` — Facial recognition accuracy disparities across demographics
- `historical_comparison.png` — Side-by-side comparison of industrial and algorithmic exploitation
