# Article 03 — Marx's Alienation in the Algorithm Age

**From factory floors to feed algorithms: is Marx's alienation theory more relevant than ever?**

Using Python to compare historical and modern alienation patterns, track alienation trends over time, and measure the autonomy gap between industrial and digital workers.

## Quick Start

```bash
pip install numpy pandas matplotlib
python alienation_analysis.py
```

## What You'll Get

**Charts:**
- `alienation_comparison.png` — Historical vs modern alienation patterns
- `alienation_trend.png` — Alienation trend over time
- `autonomy_gap.png` — Worker autonomy gap: industrial vs digital era
