# SEO Titles & Meta Descriptions (zh-TW · en · ja)

> **Apply in:** WordPress → edit each post → **Yoast SEO** box → *SEO title* and
> *Meta description* fields (set them on **each language version** of the post).
> Don't paste the Yoast title into the H1 — keep the on-page H1 as the natural
> headline; the *SEO title* is what shows in Google.

**Length targets** (so Google doesn't truncate):
- **SEO title:** ≤ ~60 Latin chars / ≤ ~30 full-width CJK chars. Lead with the
  keyword, end with `| Code & Cogito`.
- **Meta description:** ~140–155 Latin chars / ~70–80 full-width CJK chars.
  One sentence of intent + one of payoff. Each language is written for *that
  market's* search intent — **do not machine-translate one into the others.**

Every description ends, in effect, with a reason to click (data, code, the
counter-intuitive finding). CJK copy omits the brand suffix to save space.
