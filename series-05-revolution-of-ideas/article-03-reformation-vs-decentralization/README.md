# Article 03 — Reformation vs Decentralization Movement

**Martin Luther (1517) vs Satoshi Nakamoto (2008): when authority crumbles, who fills the void?**

Using Python to model authority hierarchy collapse, simulate thesis propagation, visualize church division networks, compare indulgences to ICOs, measure decentralization (Nakamoto coefficient), and simulate phase transitions.

## Quick Start

```bash
pip install numpy matplotlib networkx
python reformation_vs_decentralization_analysis.py
```
