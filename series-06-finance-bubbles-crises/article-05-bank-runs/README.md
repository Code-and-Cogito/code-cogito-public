# Article 05 — Bank Runs: Northern Rock to Silicon Valley Bank

**2007 vs 2023: bank runs got faster, but the game theory never changed.**

Using Python to model maturity mismatch, compare bank run speeds (physical vs digital), simulate Nash equilibrium in depositor decisions, evaluate deposit insurance effectiveness, analyze SVB's collapse, and model financial contagion.

## Quick Start

```bash
pip install numpy matplotlib
python bank_runs_analysis.py
```
