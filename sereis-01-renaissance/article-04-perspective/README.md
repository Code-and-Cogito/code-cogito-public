## 核心公式

透視投影的數學本質：

```
x' = (d × x) / z
y' = (d × y) / z
```

**關鍵概念：**
- 除以深度 z → 距離越遠，投影越小
- 這就是「近大遠小」的數學表達
- 所有平行線在投影後交會於消失點

## 關鍵發現

**透視法的革命性意義：**
- 把繪畫從「手藝」提升到「科學」
- 藝術家可以精確計算每個點的位置
- 不再只憑直覺，而是用數學驗證

**一點透視法特徵：**
- 一個消失點（通常在畫面中心）
- 適合表現街道、走廊、鐵軌等場景
- 文藝復興早期最常用的技法

**歷史影響：**
- 改變了整個藝術史
- 啟發工程製圖與建築設計
- 現代3D圖學的數學基礎
- 電影、遊戲產業的核心技術

## 程式架構

### 核心函數

**透視投影：**
```python
def perspective_projection(point_3d, d=5.0):
    """將3D點投影到2D平面"""
    x, y, z = point_3d
    x_proj = (d * x) / z
    y_proj = (d * y) / z
    return x_proj, y_proj
```

## 技術細節

**座標系統：**
- 觀察者位於原點 (0, 0, 0)
- Z軸正方向為深度（遠離觀察者）
- 畫面平面位於 z = d 處

**參數說明：**
- `d`: 畫面平面距離（預設 5.0）
- `size`: 立方體邊長
- `center`: 立方體中心位置 (x, y, z)
