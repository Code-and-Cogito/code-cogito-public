# Article 03 - 達文西解剖學分析

## 這是什麼？

用Python重現達文西的維特魯威人研究，驗證人體的黃金比例，視覺化完美的人體比例。

## 快速開始

### 安裝需求

```bash
pip install numpy matplotlib scipy
```

### 運行程式

```bash
# 維特魯威人分析
python vitruvian_man_analysis.py

# 黃金比例驗證
python golden_ratio_verification.py
```

## 輸出

**程式一：維特魯威人分析**
- 終端機：人體比例一覽表、黃金比例驗證、關鍵發現
- 圖片：`vitruvian_man.png` - 圓形+正方形兩種構圖

**程式二：黃金比例驗證**
- 終端機：黃金比例數學特性、費波那契數列、應用範例

## 關鍵發現

**人體黃金比例：**
- 肚臍分割比例：φ = 1.618...（黃金比例）
- 身高 = 8個頭長
- 雙臂張開 = 身高
- 手掌長 = 臉長
- 腳長 = 身高的 1/7

**達文西的發現：**
- 人體不是隨機的，遵循數學規律
- 肚臍是黃金分割點
- 圓形與正方形可以完美內接人體
- 數學即美學，人體即宇宙

## 維特魯威人的兩種構圖

**圓形構圖：**
- 雙臂雙腿張開
- 以肚臍為圓心
- 手腳觸碰圓周

**正方形構圖：**
- 雙腿併攏，雙臂平舉
- 高度 = 寬度 = 身高
- 形成完美正方形
