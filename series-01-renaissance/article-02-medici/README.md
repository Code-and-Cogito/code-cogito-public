# Article 02 — Medici Bank Network Analysis

**The Medici Family: A Financial Empire's Cultural Investment**

Using Python and NetworkX to reconstruct the Medici Bank's European branch network (1397-1494) and analyze their hub-and-spoke financial empire.

## Quick Start

```bash
pip install networkx matplotlib numpy
python medici_bank_analysis.py
```

## What You'll Get

**Terminal output:**
- Branch revenue rankings
- Network statistics and structure analysis
- Hub-and-spoke pattern detection
- Bill of exchange profit calculations

**Chart:**
- `medici_bank_network.png` — Bank network visualization with geographic positioning

## Key Findings

**Medici Bank Network:**
- 8 branches across Europe
- 97 years of operation (1397-1494)
- Hub-and-spoke structure centered on Florence
- Total annual revenue: ~405,000 florins

**Financial Innovation:**
- Bills of exchange yielded ~16.67% annualized returns
- Interest disguised as "exchange rate differences" to circumvent usury laws
- An early form of financial engineering
