# Article 05 — Humanism Network Analysis

**Humanism — The Birth of Human Dignity**

Using Python to map the intellectual network of Renaissance humanists and visualize the paradigm shift from God-centered to human-centered thinking.

## Quick Start

```bash
pip install numpy matplotlib
python humanism_analysis.py
```

## What You'll Get

**Terminal output:**
- Most connected thinker and influence scores
- Key turning point identification

**Charts:**
- `humanist_network.png` — Renaissance humanist influence over time (bubble size = connections)
- `paradigm_shift_radar.png` — Radar chart: Medieval God-centered vs Humanist human-centered worldview
- `text_discovery_timeline.png` — Timeline of key classical text rediscoveries that fueled humanism

## Key Findings

**Highest betweenness centrality (intellectual bridges):**

| Thinker | Role |
|---------|------|
| Petrarch (1340) | Bridge between classical antiquity and early humanism |
| Ficino (1462) | Brought Plato into the Italian mainstream |
| Erasmus (1500) | Connected Italian humanism to Northern Europe |

The paradigm shift radar shows a near-complete inversion across all five dimensions — from God-centered scores of 85-95% to human-centered scores of 15-35%. This wasn't gradual erosion; it was a civilizational flip.
