# Article 01 — Florence Network Analysis

**Why Florence Became the Cradle of the Renaissance**

Using Python and NetworkX to reconstruct the 15th-century Italian trade network and analyze how Florence became the hub of the Renaissance.

## Quick Start

```bash
pip install networkx matplotlib numpy
python florence_network_analysis.py
```

## What You'll Get

**Terminal output:**
- Network statistics (nodes, edges, density)
- Betweenness centrality ranking
- Florence vs competitors comparison

**Chart:**
- `florence_network_basic.png` — Trade network visualization with node sizes proportional to centrality

## Key Findings

Florence's betweenness centrality was:
- **2.2x** higher than Venice
- **1.8x** higher than Milan
- **9.9x** higher than Rome

More information, capital, and talent flowed through Florence than any other Italian city — explaining why it became the cradle of innovation.
