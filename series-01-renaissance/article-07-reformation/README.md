# Article 07 — Reformation Analysis

**Reformation — When Authority Crumbles**

Using Python to map the timeline of the Reformation, visualize the collapse of Church authority, and quantify 131 years of religious conflict.

## Quick Start

```bash
pip install numpy matplotlib
python reformation_analysis.py
```

## What You'll Get

**Terminal output:**
- Duration of upheaval (131 years)
- Total estimated deaths across religious wars
- Largest single conflict identification

**Charts:**
- `reformation_timeline.png` — Key events from Luther's 95 Theses (1517) to Peace of Westphalia (1648)
- `authority_radar.png` — Church authority before and after the Reformation
- `religious_wars_deaths.png` — Death toll across major religious conflicts

## Key Findings

- The Reformation triggered **131 years** of upheaval (1517-1648)
- The Thirty Years' War alone killed an estimated 8 million people
- Church authority collapsed across every measured dimension
