# Article 08 — Enlightenment Analysis

**Enlightenment — When Reason Becomes the New God**

Using Python to map the intellectual network of the Enlightenment, compare reason vs tradition across domains, and trace how ideas became revolutions.

## Quick Start

```bash
pip install numpy matplotlib
python enlightenment_analysis.py
```

## What You'll Get

**Terminal output:**
- Most influential Enlightenment thinker
- Movement span in years
- Key philosophical foundation for the American Revolution

**Charts:**
- `enlightenment_thinkers.png` — Key thinkers: influence scores and connections
- `reason_vs_tradition.png` — Reason vs tradition dominance across domains
- `ideas_to_revolutions.png` — How Enlightenment ideas fueled political revolutions

## Key Findings

- Newton scored highest in influence — his method became the template for all Enlightenment thinking
- Locke's Natural Rights theory (score: 90) was the philosophical foundation for the American Revolution
- The movement spanned ~94 years (1687-1781)
