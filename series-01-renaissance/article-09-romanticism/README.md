# Article 09 — Romanticism Analysis

**Romanticism: When Poets Rebel Against Philosophers**

Using Python to analyze the themes, timeline, and cross-domain influence of the Romantic movement — the emotional backlash against Enlightenment reason.

## Quick Start

```bash
pip install numpy matplotlib
python romanticism_analysis.py
```

## What You'll Get

**Terminal output:**
- Most common Romantic theme
- Movement span (~66 years)
- Number of art forms covered

**Charts:**
- `enlightenment_vs_romanticism.png` — Value comparison: reason vs emotion across dimensions
- `romantic_works_timeline.png` — Major Romantic works across literature, music, and art
- `romantic_themes.png` — Theme frequency analysis across Romantic works

## Key Findings

- "Nature Sublime" was the most frequent theme in Romantic works
- The movement spanned ~66 years (1774-1840) across literature, music, and painting
- Romanticism was not anti-science — it was anti-reductionism
